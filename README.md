## üí° Introduction

Realworld: "The mother of all demo apps" ‚Äî Exemplary back-end Medium.com clone (called [Conduit](https://github.com/yoonge/conduit-axum)) in Rust, built with Axum + jsonwebtoken + PostgreSQL + SQLx, it provides RESTful APIs for [Conduit React](https://github.com/yoonge/conduit-react).


## üî∞ Getting Started

```sh
$ git clone https://github.com/yoonge/conduit-axum.git

$ cd conduit-axum

# Change `.env` file with your own username, password, database_name etc.
$ cp .env.example .env

$ cargo install sqlx-cli

$ sqlx migrate run

$ cargo run
```
